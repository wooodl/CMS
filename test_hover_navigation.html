导航栏悬停功能测试页面

测试左侧导航栏收缩状态下的鼠标悬停显示子菜单功能

测试步骤：

  1. 访问主应用程序：https://localhost:5000
  2. 点击左上角的菜单按钮（三条横线图标）收缩左侧导航栏
  3. 将鼠标悬停在有子菜单的图标上（如人员管理、飞机管理等）
  4. 观察子菜单是否在右侧正确显示
  5. 移动鼠标到子菜单上，确认子菜单保持显示
  6. 移开鼠标，确认子菜单正确隐藏
  7. 测试不同菜单项的悬停效果

改进后的功能特性：

✓ 平滑过渡动画：子菜单显示和隐藏有 opacity 和 transform 过渡效果
✓ 智能延迟控制：鼠标进入有 100ms 延迟，避免快速移动时频繁显示
✓ 用户友好的隐藏延迟：鼠标离开后有 300ms 延迟再隐藏，避免误操作
✓ 视觉指示器：子菜单左侧有三角形箭头指示器，清晰显示关联关系
✓ 美观的样式：子菜单有阴影、圆角和边框，视觉效果更佳
✓ 悬停高亮：菜单图标和子菜单项都有悬停高亮效果
✓ 完全隐藏下拉箭头：收缩状态下不显示任何下拉指示器
✓ 状态记忆：页面刷新后记住导航栏的展开/收缩状态
